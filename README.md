A simple browser-based car game built using HTML, CSS, and JavaScript.  
The player controls the car using keyboard arrow keys and tries to score maximum points while avoiding obstacles.

## Features
- Smooth keyboard controls
- Score tracking system
- Clean and responsive UI
- Beginner-friendly JavaScript logic

## Technologies Used
- HTML
- CSS
- JavaScript

## How to Play
1. Open the game in a browser.
2. Click on **Start**.
3. Use **Arrow Keys** to move the car.
4. Avoid obstacles and achieve the highest score.

## Purpose
This project was created to practice JavaScript game logic, DOM manipulation, and keyboard event handling.
